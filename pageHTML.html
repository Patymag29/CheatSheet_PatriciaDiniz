TIPS - Best Practices

  • Keep your code commented, organized, and correctly indented.
  • Prefer to use an external JavaScript file to include all logic for your page.
  • Prefer to style your HTML page using an external CSS stylesheet.
  • Prefer to split and organize your JavaScript code using functions.
  • Create variables and constants with names that facilitate better understanding, and comment on their meaning.
  • When creating new code, think about others who may maintain or remodel it in the future. Build your code as clean and comprehensive as possible.
